# Python图书管理系统 python6

## 项目概述

本项目是一个基于Python语言的图书管理系统，提供了一套完整的源码、运行教程、开发文档及PPT，旨在为图书馆或个人藏书管理提供便捷的数字化解决方案。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- 用户模块
  - 登录
  - 注册
- 图书信息管理
  - 添加
  - 修改
  - 查询
  - 删除
- 图书借阅记录
- 读者信息查询

## 系统角色

- 系统管理员
- 读者

## 运行环境

- Python环境
- Flask框架
- MySQL数据库

## 部署步骤

1. 配置Python环境
2. 安装Flask及其他依赖
3. 配置MySQL数据库
4. 运行项目

## 目录结构

```
project/
│
├── app/              # Flask应用目录
│   ├── models/       # 数据模型
│   ├── views/        # 视图处理

│
├── docs/             # 开发文档
├── run.py            # 项目启动脚本
└── requirements.txt  # 依赖列表
```
